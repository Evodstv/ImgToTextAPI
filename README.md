# ImgToText API

## Описание проекта

**ImgToText API** — веб-сервис для распознавания текста на изображениях.

---

## Целевая аудитория

Сервис предназначен для **разработчиков**.  
Он позволяет получать текст с изображений без необходимости самостоятельно настраивать и использовать OCR-движки.

---

## Основные функции API

- **Проверка доступности сервиса**  
  Позволяет убедиться, что API работает и готово принимать запросы.

- **Распознавание текста на изображении**  
  Сервис принимает изображение с текстом, выполняет OCR-распознавание и возвращает результат в виде строки.  
  Поддерживается выбор языка распознавания:  
  `rus`, `eng`, `rus+eng`.

---

## Архитектура и используемые технологии

- **Язык программирования:** C#  
- **Платформа:** ASP.NET Core Web API (.NET 10)  
- **OCR-движок:** Tesseract OCR  
- **Документация API:** Swagger
